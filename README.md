# 🏥 Health VIP – Smart Healthcare Web Portal

A modern **Single Page Application (SPA)** healthcare portal interface designed to simulate a premium medical service experience.
This project focuses on **UX, modern UI effects, and interactive web functionality using Vanilla JavaScript**.

---

## 🌐 Live Demo

https://162216.github.io/health-vip-portal/

---

## 🚀 Features

### 🧠 Smart Chatbot Assistant

* Keyword recognition
* Auto navigation to relevant pages
* Typing animation simulation

### 🔍 Intelligent Search Engine

* Semantic search logic
* Medical department matching (e.g. *KBB → ENT*)
* Modal feedback system

### 🌙 Dark Mode

* Theme switching
* Saved using **LocalStorage**
* Instant UI updates with CSS variables

### ⚡ SPA Navigation

* No page reloads
* Section based routing
* App-like user experience

### 📱 Fully Responsive Design

* Desktop optimized layout
* Tablet adaptive structure
* Mobile off-canvas navigation

### 🎨 Modern UI Effects

* Glassmorphism header
* Parallax hero section
* Interactive hover effects
* Smooth transitions
